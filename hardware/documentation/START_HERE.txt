CRITICAL INFORMATION (READ THIS)
================================================================================

I2C ADDRESSES (Must be correct):
  ADS1115 #1 (Left Foot):  0x48 ← ADDR pin tied to GND
  ADS1115 #2 (Right Foot): 0x49 ← ADDR pin tied to 3.3V

KEY COMPONENT PINOUTS:
  ESP32 GPIO21 = I2C SDA
  ESP32 GPIO22 = I2C SCL
  ESP32 GPIO0  = BOOT button
  ESP32 EN     = RESET button

VOLTAGE LEVELS:
  USB Input: 5V
  Logic Power: 3.3V
  All signals: 3.3V (NOT 5V)

CONNECTORS:
  USB-C: Power + UART programming
  JST-PH (6×): FSR sensors (one per sensor location)
  JST-PH (1×): Battery (future, not populated)

CRITICAL RESISTORS:
  R10, R11: 4.7kΩ I2C pull-ups (required)
  R1-R6: 10kΩ FSR voltage dividers
  R7: 1kΩ LED current limiting
  R12, R13: 10kΩ button pull-ups

COMMON QUESTIONS ANSWERED
================================================================================

Q: Do I need to use JLCPCB?
A: No, you can use PCBWay or any PCB manufacturer. JLCPCB is recommended
   for lowest cost and best LCSC component availability.

Q: Can I modify the design?
A: Yes, absolutely. The schematic and netlist are provided for customization.
   Review PCB_DESIGN.md before making changes.

Q: How many boards should I order?
A: Minimum 5 from JLCPCB (economies of scale). First prototype = 5, then
   order more for production.

Q: What if parts are unavailable?
A: BOM.csv includes alternates. Key alternates:
   - CP2102N ↔ CH340C (USB to UART)
   - AMS1117 ↔ SGM1117 (voltage regulator)
   - ADS1115 ↔ ADS1015 (if lower resolution acceptable)

Q: What tools do I need for assembly?
A: Minimum: Soldering iron, solder, tweezers, multimeter
   Optional but helpful: Hot air station, reflow oven, magnifying glass

Q: How long does it take to assemble?
A: 4-8 hours for hand assembly (80+ solder joints)
   1-2 hours if using reflow oven

Q: What's the cost breakdown?
A: Components: $12-18 per board
   PCB (5 qty): $1-2 per board
   Shipping: Varies by location
   TOTAL: $15-25 per board (not including shipping)

Q: Can I make this wireless without BLE?
A: Not with this design. Firmware could be modified for WiFi instead,
   but that requires ESP32 changes and antenna modifications.

Q: What's the maximum distance for BLE?
A: Typical: 10-30 meters (open space)
   With good antenna: 50-100 meters possible
   Review ESP32 datasheet for antenna options

Q: Is this waterproof?
A: No, but can be made water-resistant with enclosure + conformal coating.
   Sensors themselves should be potted/sealed separately.

Q: Can I add a battery?
A: Yes, J8 connector reserved for battery. Battery charging circuit needed
   (not included in this design). See PCB_DESIGN.md Section 9 for v2.0 ideas.
